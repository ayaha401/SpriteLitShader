# SpriteLitShader

# 概要
SpriteRendererから影が落ちるシェーダー

# 動作環境
Unity 2020.3.8f1

# 機能
* 影を落とす


# 注意事項
* SpriteRendererに設定が必要。詳しくはこちら<br>
https://github.com/ayaha401/SpriteLitShader/wiki/SpriteRenderer%E8%A8%AD%E5%AE%9A
